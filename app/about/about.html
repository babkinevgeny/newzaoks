SET SERVICE GROUP – Центр металлообработки в Санкт-Петербурге, основанный в 2015году для реализации по налаживанию производственных связей в рамках программы по развитию производственного потенциала России.

«Мы – это команда специалистов в различных видах металлообработки, объединенные одной большой идеей, развитие производственного сектора России и качественное увеличение номенклатуры товаров, произведенных на территории России.»

Основные направления деятельности SET SERVICE GROUP:

  1. Изготовление деталей, отдельных узлов различных механизмов для машиностроения из разных материалов – сталь, нержавейка, сплавы цветных металлов по чертежам заказчика;
  2. Изготовление сварных и сборочных конструкций;
  3. Поиск и подбор партнеров по отдельным видам технологических операций. Инжиниринг. Инновационная деятельность. Разработка и внедрение новых технологий, конструкций, оснастки, приспособлений;
  4. Разработки и испытания в области специальных сталей и сплавов при поддержке Национального исследовательского центра «Курчатовский институт»;
  5. Поиск решений, обеспечивающих импортозамещение по таким отраслям как: нефтегазовая промышленность, машиностроение, мостостроение и т.д.

SET SERVICE GROUP осуществляет единичное и серийное производство оборудования, разработанного как в SET SERVICE GROUP, так и сторонними разработчиками. Мы располагаем производственной базой обрабатывающими центрами, станками с ЧПУ, парком токарных и фрезерных станков, прочим металлообрабатывающим оборудованием.

Так же мы развиваем сеть производственного субконтрактинга по всей России, что позволяет нам:

  1. Задействовать уникальное оборудование по России для выполнения самых сложно реализуемых проектов;
  2. Выполнять заказы в самые сжатые сроки;
  3. Выполнять крупные и серийные заказы;
  4. Брать на реализацию проекты «под ключ».

Качество продукции контролируется в соответствии с конструкторской документацией, в том числе с помощью различных видов испытаний, куда входят все виды неразрушающего контроля, пневмо- и гидроиспытания, испытания на прочность. Компания включена в общий реестр организаций имеющих сертифицированную систему менеджмента качества по требованиям международного стандарта ISO 9001:2015.
